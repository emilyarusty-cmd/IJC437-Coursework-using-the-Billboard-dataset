# IJC437 Coursework – Billboard Chart Success

This repository contains the analysis for the IJC437 Introduction to Data Science coursework.

The project investigates whether Spotify audio features can explain and predict song success on the Billboard Hot 100 between 2000 and 2023.

## Research Questions

1. Which Spotify audio features are most strongly associated with peak Billboard chart position?
2. How have the audio characteristics of Top 10 songs changed over time?
3. Can Top 10 chart success be predicted using Spotify audio features?

## Repository Structure

- `billboard_analysis.R` – Main analysis script
- `billboard_24years_lyrics_spotify.csv` – Input dataset
- `figures/` – Figures generated by the analysis
- `tables/` – Tables generated by the analysis
- `docs/` – GitHub Pages site
